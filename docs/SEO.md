# SEO Design Notes

## Current SEO implementation

- Dedicated tools index page: `/tools`
- Dedicated category pages: `/categories/[slug]`
- Tool detail pages generated from a known registry: `/tools/[slug]`
- Per-page metadata: title, description, canonical, Open Graph, Twitter card
- Structured FAQ sections on tool pages
- Internal linking between related tools and categories
- `robots.txt` and `sitemap.xml` generation

## Content conventions

- Clean stable slugs
- Descriptive H1 and intro copy per page
- Helpful FAQ copy (not keyword stuffing)
- Avoid thin pages

## Future SEO expansion

- Add `/guides` and `/blog`
- Build topic clusters around each tool
- Add comparison and troubleshooting pages
